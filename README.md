# Desafio Avanti

Este repositório contém o código fonte para o Desafio de Programação do Bootcamp Avanti.

## Pastas

### 1. SQL

A pasta "SQL" contém os scripts SQL relacionados ao desafio.

- *`Criar_DB.txt`*: Um breve passo a passo de como criar e inicializar o banco de dados.
- *`eventos.sql`*: Script para criar a tabela eventos.
- *`categorias.sql`*: Script para criar a tabela categorias.
- *`locais.sql`*: Script para criar a tabela locais.

### 2. APP

A pasta "APP" contém o código-fonte da aplicação relacionada ao desafio.
